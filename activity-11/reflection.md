
Key Learnings: 
- I learned a more detailed definition of state and how react moniters it and updates the DOM
- React creates functions for you if you want, so you don't have to write the function definition
- Everything related to one componant is written inside that component
- I learned that props are basically just read-only parameters
- You can use your created components in the HTML instead of creating a generic div

Comparison:
- There is a lot less code in the react version
- There is no manually manipulating the DOM elements
- In react there are no explicit .addEventListeners. Instead you assign a function to be executed onClick when you're making the component or in the HTML
- Any changes made to the board are done automatically in react, you don't need to manually add or remove things
